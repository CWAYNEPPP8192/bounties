# Gaia Bounties

Welcome to the Gaia Bounties repository! 🚀

This repo hosts open-source contribution opportunities for Gaia. Developers, writers, and community members can participate in bounties, submit PRs, and help improve the Gaia ecosystem.

## How It Works

1. **Check Open Bounties** – Browse the [Issues](https://github.com/GaiaNet-AI/bounties/issues)
2. **Claim a Task** – Comment on the issue if you're interested
3. **Complete & Submit** – Follow the bounty requirements and open a PR
4. **Earn Rewards** – Once merged, the bounty will be distributed accordingly
